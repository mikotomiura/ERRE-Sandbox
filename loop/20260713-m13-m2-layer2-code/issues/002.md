# Issue 002 (J2): build_self_other_context pure builder（strict prefix filter・float-free）
verify_level: recheck

## Goal
`build_self_other_context(*, observer_id, window_index, prior_records) -> SelfOtherContext` を
society.py に **pure 決定関数** として実装（v1 graft）。他者 prior-window 記録行動 → canonical
float-free payload + 決定的 render text segment。**strict prefix filter**
（`other_agent_id != observer_id` ∧ `source_window < window_index`）で self/future leak を構造排除。

## Background
契約 §L3（pure builder 行）+ §L7（payload schema）+ DA-L2-2（strict prefix filter）+ Codex HIGH-2。
payload = §L7 の構造（injections[] canonical-sorted by (window, observer_agent_id)、observed[]
canonical-sorted by other_agent_id、scalar は {str,int,bool,None} のみ float 排除）。render は
payload の pure 関数、SimToM framing template（functional-analog 語彙、bounded）。
※ SimToM 最終文言 + think=False parseability desk-audit は J4 に集約（builder は render 骨格のみ、
文言 const は J4 desk-audit で確定）。ここでは builder purity + strict prefix filter を確立。
