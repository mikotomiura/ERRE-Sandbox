# Issue 001 (J1): build_user_prompt に self_other_context kwarg + seam thread（byte-invariant）
verify_level: recheck

## Goal
observer の既存 cognition call に self-other context を注入できる **seam** を additive に開通する。
`build_user_prompt` に keyword-only `self_other_context` を追加（M10-B `world_model_entries`
idiom = 空なら byte-identical・USER 側・memory 非書込み）、`cognition/cycle.py:step` →
`world/tick.py:step_cognition_once` → `_step_one` へ transient に thread（default None）。

## Background
契約 §L3 seam 表 + §L11 修正リスト。M10-B `world_model_entries`（prompting.py L333/L348-353）が
「空なら byte-identical・USER 側・memory 非書込み」の確立 idiom。self-other は同型 kwarg。
Layer2 empty → 完全 byte-identical（既存 golden/N=1 不変）。cycle.step の build_user_prompt 呼出
（cycle.py L688-693）に kwarg を渡すのみ。
