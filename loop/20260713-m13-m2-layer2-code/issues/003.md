# Issue 003 (J3): run_society_loop Layer2 param（default-off）+ run-level slot 集約
verify_level: recheck

## Goal
`run_society_loop` に Layer2 param（`self_other_enabled: bool = False` 等 additive、default-off）を
追加。enabled 時、各 window で builder を呼んで observer 毎の self-other context を作り
step_cognition_once へ thread（J1 seam 経由）、全 (observer, window) の注入を **run-level に集約**した
canonical payload を `self_other_observation_input` slot に populate（off/N=1/all-empty → None）。

## Background
契約 §L3 slot populate 行 + §L7 run-level 集約 + §L8 N=1/off byte 不変 + Codex MEDIUM-5（4 経路分割）
/ HIGH-3（determinism 継承 serializer）。Layer1 は slot 常時 None（society.py L1061）。逐次 sorted
scheduler ループ（L1016-1052）は **無改変**、Layer2 は additive step（default-off でスキップ）。
slot payload は event_log_checksum の同一 json.dumps に埋込まれる（継承 serializer、独自 serializer 無）。

## Scope
### In
- `run_society_loop(..., self_other_enabled: bool = False)` additive param。
- enabled 経路: window t で各 observer に builder（prior window の他者 records）→ context を
  step_cognition_once へ thread（J1 seam）+ run-level slot 集約（empty→None）。
- prior-window records の収集（Layer1 が既に持つ rows/dialog_events/pair_events 由来、read-only）。
### Out
- builder 本体（J2 完了前提）、prompt seam（J1 完了前提）、continuity gate（J4）、guard/golden（J5）。
- 逐次 sorted scheduler ループの構造改変（additive step のみ、ループ本体は無改変）。

## Allowed Files
- `src/erre_sandbox/integration/embodied/society.py`（run_society_loop Layer2 param + slot 集約 additive）
- `tests/test_integration/test_m2_society_selfother.py`（J3 分の test 追記）

## Acceptance Criteria（AC↔test）
- **AC1** slot が期待 canonical payload で populated iff（enabled ∧ 他者存在 ∧ window≥1）；
  off/N=1/all-empty は None。
  → `test_self_other_slot_provenance`（#4）
- **AC2** N=1/off/empty 各経路（default-omitted / flag-off / N=1-enabled-empty / N≥2-all-empty）→
  slot None・空 header なし → Layer1 と byte-equivalent。`null` slot header/空 payload header が出たら違反。
  → `test_self_other_n1_degenerate`（#9）
- **AC3** populated slot → event_log_checksum が同一 (seed, 記録 Plane2) の replay で byte 一致 +
  payload float-free + 継承 serializer ゆえ cross-platform byte 一致。
  → `test_self_other_event_log_checksum_stable`（#8）

## Test Plan
